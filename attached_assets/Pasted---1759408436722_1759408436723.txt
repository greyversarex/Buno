**Твоя задача — реализовать безопасное и унифицированное отображение изображений по всему сайту.**

1.  **Создай "Золотую" Функцию.** В общем JS-файле (например, `frontend/public/js/utils.js`, создай его, если не существует и подключи ко всем страницам) создай следующую функцию:

    ```javascript
    function getAbsoluteImageUrl(relativePath) {
      const placeholder = '/путь/к/твоей/картинке-заглушке.jpg'; // УКАЖИ ЗДЕСЬ РЕАЛЬНЫЙ ПУТЬ К ЗАГЛУШКЕ
      if (!relativePath || typeof relativePath !== 'string') {
        return placeholder;
      }
      if (relativePath.startsWith('http')) {
        return relativePath; // Уже абсолютный путь
      }
      if (relativePath.startsWith('/')) {
         // Безопасно создаем URL из корня сайта
        return new URL(relativePath, window.location.origin).href;
      }
      // Если путь пришел без слеша в начале, например "uploads/...", добавляем его
      return new URL('/' + relativePath, window.location.origin).href;
    }
    ```
